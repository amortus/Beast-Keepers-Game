# Skybox para Cena de Batalha

Coloque aqui a imagem da skybox equirectangular (360°).

**Nome do arquivo:** `battle-skybox.jpg` ou `battle-skybox.png`

**Formato:** Equirectangular (panorama 360°)

**Tamanho recomendado:** 2048x1024 ou 4096x2048 pixels

O código tentará carregar primeiro `.jpg`, depois `.png` se o `.jpg` não existir.
